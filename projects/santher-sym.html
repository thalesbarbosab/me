Santher Sym - Institucional

  • Tipo: Institucional
  • Clientes: Webeleven / Santher
  • Data do projeto: Dezembro 2021 a Março 2023
  • Tecnologias utilizadas
  • PHP Laravel Nginx MySQL AWS S3 AWS SNS Redis AWS Load Balance AWS EC2 Bootstrap CSS HTML JavaScript API jQuery GitHub Google Analytics

Sobre o projeto

A Santher procurou a Webeleven para desenvolvimento de um website para exibir os produtos da marca Sym, e auxiliar seus consumidores com um calendário menstrual completo (com mais informações que dos principais concorrentes) com envio por e-mail dos ciclos principais, artigos para leitura, exibição do catalogo de produtos, etc.

Minhas responsabilidades

- Desenvolvimento backend de todas as funcionalidades da aplicação seja relatórios, ambiente cms para gestão dos conteúdos do site, testes das funcionalidades;
- Auxílio para a equipe de frontend e criação para sugestões de layout além de integração do backend com o frontend;
