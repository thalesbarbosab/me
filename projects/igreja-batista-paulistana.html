Sobre o projeto

Dentro da organização religiosa Igreja Batista Paulistana houve a necessidade de modernização e atualização do website da igreja para uma tecnologia mais atual, segura e paginas mais modernas para que o visitante pudesse acessar e conhecer mais sobre. Então surgiu esse projeto voluntário para desenvolvimento.

Desenvolvido uma frente website e também uma frente CMS para alteração dos conteúdos, como banners, textos, informações de contato, projetos grupos, pastores, notícias, mensagens, dados bancários para informações de contribuição, etc.

Minhas responsabilidades

Em contato com a equipe de mídia da igreja, fiquei responsável de todo o planejamento de construção do software desde a criação das funcionalidades, desenvolvimento do frontend, desenvolvimento backend até a entrega em ambiente de produção.
